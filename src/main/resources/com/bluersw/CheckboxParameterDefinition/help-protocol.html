Protocol for obtaining remote files, such as http and https. If you select HTTP or HTTPS, please fill in the URL of the file in the URI, for example: https://raw.githubusercontent.com/sunweisheng/Jenkins/master/xxx.yaml
JENKINS_LOCAL is the Jenkins server file system. If you choose JENKINS_LOCAL, please fill in the file path on the Jenkins server in the URI, such as: /opt/xxx.yaml or C:\xxx.yaml
